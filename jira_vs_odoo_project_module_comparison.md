# Jira vs Odoo Project Sprint Management Module Comparison

---

## 1. Overview

This document compares the customized **Odoo Project Sprint Management Module** with **Jira Project Management System**.

The purpose of this comparison is to evaluate implemented project management features and compare the module with enterprise-level agile project management software.

The customized Odoo module has been enhanced with sprint management, backlog handling, subtasks, reporting, burndown tracking, role-based permissions, notifications, attachments, and advanced workflow management.

Current implementation level is approximately **85–90% similar to Jira**.
