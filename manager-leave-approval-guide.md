# Manager Leave Approval 

## 1. Purpose of This Document

This document guides managers on how to review, approve, and refuse employee leave requests in Odoo 19 Community Edition.

---

## 2. Manager Role Overview

Managers must:

- Review leave requests  
- Ensure operational coverage  
- Follow company HR leave policies  

---

## 3. Accessing Leave Requests

Go to:

**Time Off → Approvals**

Here you can view:

- Pending requests  
- Approved requests  
- Refused requests  

---

## 4. Approving a Leave Request

1. Navigate to **Time Off → Approvals**  
2. Open the pending request  
3. Review:
   - Leave type  
   - Dates  
   - Leave balance  
4. Click **Approve**  
5. Employee receives notification  

---

## 5. Refusing a Leave Request

1. Open the leave request  
2. Click **Refuse**  
3. Add rejection reason (recommended)  
4. Employee receives notification  

---

## 6. Calendar View

Go to:

**Time Off → Calendar**

- View all employee leaves  
- Identify overlapping leaves  
- Understand leave distribution  

---

## 7. Manager Dashboard Insights

Dashboard provides:

- Pending approvals  
- Monthly leave summary  
- Employee-wise leave data  

---

## 8. Best Practices

- Approve or refuse requests promptly  
- Check leave balance before approval  
- Use calendar to avoid overlapping leaves  
- Provide clear reasons for refusal  

---

✅ Manager leave approval process completed successfully.

## Conclusion

Managers can efficiently handle employee leave approvals in Odoo using structured workflows and proper validation.
