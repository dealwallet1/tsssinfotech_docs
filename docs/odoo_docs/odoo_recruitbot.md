# RecruitBot Documentation

## Overview
RecruitBot is an AI-powered recruitment assistant that helps filter candidates based on skills, experience, and technologies.

---

## Features

### 1. Skill Filter
- Users can filter candidates by:
  - Python
  - Java
  - React
  - SQL
  - AI/ML

### 2. Candidate Dashboard
- Displays total candidates
- Shows skill-wise distribution

### 3. AI Chatbot
- Ask queries like:
  - "Show Python candidates"
  - "Show React developers"

---

## Fields Explanation

### Candidate Model Fields

| Field Name | Type | Description |
|-----------|------|------------|
| name | Char | Candidate name |
| email | Char | Email address |
| skills | Text | Candidate skills |
| experience | Integer | Years of experience |
| location | Char | Candidate location |

---

## Dashboard

- Skill cards show candidate count
- Clicking card filters candidates
