multiagent connection  by using agno agents

1.Multiple agents are connected and grouped into a team.
2.This team is connected to AgentOS, where we enable the A2A (Agent-to-Agent) interface.
3.Once A2A is enabled, the team is exposed through a server URL like:
http://localhost:7777/a2a/teams/doctor-appointment-team
4.Using this server URL, we integrated the system with Odoo by utilizing the A2AClient.
5.Through this integration, Odoo can send user messages to the agent team and receive responses, enabling a chatbot-like interaction.
