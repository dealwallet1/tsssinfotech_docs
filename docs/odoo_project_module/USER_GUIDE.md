# 📘 User Guide – Odoo Project Module

## 🧾 Overview
This module helps manage projects, tasks, sprints, and workflows efficiently. Users can create tasks, track progress, assign responsibilities, monitor deadlines, and analyze productivity.

---

## ✅ Create Task

### Steps:
1. Go to **Project → Tasks**
2. Click on **New**
3. Fill in the following details:

---

## 📝 Task Details

### **Task Title**
- Provide a clear and concise name for the task  
- Example: *Fix login bug*, *Add dashboard UI*

---

### **Description**
- Explain the task in detail  
- Include:
  - Requirements
  - Expected output
  - Notes or instructions  

---

### **Assigned To**
- Select the responsible user  
- This user will receive notifications and updates  

---

### **Tags**
- Used to categorize tasks  
- Examples:
  - Bug 🐞
  - Feature 🚀
  - Enhancement ✨  
- Helps in filtering and searching tasks  

---

### **Deadline**
- Set the due date for the task  
- Used to track overdue tasks  
- Important for time management  

---

### **Project**
- Select the project under which task belongs  

---

### **Sprint**
- Assign task to a sprint (if sprint planning is used)  
- Helps in tracking sprint progress  

---

## ⏱️ Timesheets (Time Tracking)

Track time spent on tasks for productivity analysis.

### Steps:
1. Open the task  
2. Navigate to **Timesheets** section  
3. Add:
   - **Date**
   - **Time Spent (hours)**
   - **Work Description**

### Benefits:
- Helps in tracking effort  
- Useful for reporting and billing  

---

## 🔄 Workflow Stages

Tasks move through predefined stages:

1. **Backlog**
2. **To Do**
3. **In Progress**
4. **Code Review**
5. **Testing**
6. **Done**

👉 Tasks automatically move based on workflow rules  

---

## 🧑‍💻 Working on Task

- Move task to **In Progress** when work starts  
- Update task status regularly  
- Add comments for updates or issues  
- Attach files/screenshots if needed  

---

## ⏰ Deadline & Overdue Tasks

- Tasks with deadlines are monitored automatically  
- If deadline is exceeded:
  - Task becomes **Overdue**
  - Notification/activity is triggered  

---

## 🔔 Notifications & Activities

Users receive notifications for:
- Task assignment  
- Status updates  
- Overdue alerts  

Activities help track pending actions and reminders  

---

## 📊 Reports & Monitoring

Managers can:
- Track task progress  
- Analyze sprint performance  
- Monitor team productivity  

---

## 🏁 Completing Task

1. Move task to **Testing**
2. After verification → move to **Done**

---

## 💡 Best Practices

- Use meaningful task names  
- Always set deadlines  
- Update status frequently  
- Log timesheets daily  
- Use tags for better organization
